🚀 Lygron Blogger Theme

✨ Features

  • Mobile Responsive - සියලුම mobile devices වලට හරිගැහෙනවා
  • Modern Design - Glass morphism, gradient backgrounds, smooth animations
  • Fast Loading - Optimized CSS and JavaScript
  • Blogger Ready - XML template එක විදියට use කරන්න පුළුවන්
  • Custom Sections - Hero section, services, stats, blog grid

📥 Installation

  1. Download the lygron-theme.xml file
  2. Go to your Blogger Dashboard → Theme → Backup/Restore
  3. Click on "Restore" and upload the downloaded XML file
  4. That's it! Your blog will be updated with the new theme

🔧 Customization

ඔයාට ඕන විදියට මේක customize කරගන්න පුළුවන්:

  • #header-custom - Header colors and styles
  • .hero-section - Main banner content
  • .services-section - Services cards
  • .stats-section - Statistics counter
